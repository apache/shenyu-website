# shenyu .NET 语言客户端 1.0.0

### 新的功能

1. 支持 ShenYu Admin 后台与 http 类型注册到 ShenYu。
2. 支持使用 Nacos 类型注册到 ShenYu。
3. 支持使用 Zookeeper 类型注册到 ShenYu。
4. 支持使用 Consul 类型注册到 ShenYu。
5. 支持使用 Etcd 类型注册到 ShenYu。
