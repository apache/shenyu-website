ShenYu 有Casdoor插件去使用Casdoor

## 第一步.部署Casdoor

首先，我们需要部署Casdoor.

我们可以参考Casdoor官方文档去部署它.

在成功部署之后,我们需要确保:

* Casdoor服务成功在 **http://localhost:8000**上运行

* 打开你喜爱的浏览器浏览 **http://localhost:7001**,你可以看到Casdoor的登陆页面

* 输入`admin`和`123`去测试登录是否成功

  登陆成功之后，我们就可以使用casdoor.
